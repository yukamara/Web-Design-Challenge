Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, weather first pulled data from the OpenWeatherMap API to assemble a data-bs-toggle on over 500 cities.

After assembling the dataset, we used Matpotlib to plot various aspects of the weather vs. latitude. Factors we looked at include: temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
